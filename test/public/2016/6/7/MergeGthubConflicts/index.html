Deal with the github merge conflicts

First: checkout branch

bash git checkout -b <branch-name> master git pull git://github.com/{username}/{<repo>.git} master

Second: edits the conflict file

After edits in your text editor, then commit the changes:

batch git commit -a

Last: Push and merge

batch git checkout master git merge --no-ff <branch-name> git push origin master
